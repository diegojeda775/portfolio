About Me

Who am I? What do i do?

My name is Diego Alonso Ojeda Rivera. I am a full-stack software engineer, and I was born and raised in Peru. I began learning coding via Freecodecamp, then I decide to enroll in the engineering flex program at Thinkful. At Thinkful, I created many projects ranging from just frontend to just backend and finally combining both to create full-stack apps. The main technologies I used to create web apps are JavaScript, Html, Css, React, Express, Rest API’s and PostgreSQL. I also used the crud methodology when developing Rest API's to create, read, updated and delete data. I look forward to day where I can be part of a team with diverse backgrounds, so that we can push each other to be better.
